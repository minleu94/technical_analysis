# Antigravity Rule 00 - Project Entry

## 必讀入口

每次開始任務前先讀：

1. `GEMINI.md`
2. `docs/agents/README.md`
3. `docs/agents/shared_context.md`
4. `docs/agents/git_exclusions.md`
5. `docs/00_core/PROJECT_SNAPSHOT.md`

若任務涉及技術方向、Phase、優先事項或風險，再讀：

- `docs/00_core/DEVELOPMENT_ROADMAP.md` 的 Living Section
- `docs/00_core/DOCUMENTATION_INDEX.md`

## 專案定位

本專案不是單純每日吐股票名單的工具，而是可驗證、可回溯、可演化的投資決策系統。

## 目前重點

以 `docs/00_core/PROJECT_SNAPSHOT.md` 和 `docs/00_core/DEVELOPMENT_ROADMAP.md` 為準。現階段優先事項包含：

- 推薦組合回測的穩健分析指標：Sortino、Sharpe、Monte Carlo
- 評估回測最佳 Profile / Config 回灌推薦頁
- 維持 Roadmap / Snapshot / Index / UI docs / Agent docs 一致

## 語言

所有回覆、文檔、程式碼註解均使用繁體中文。
