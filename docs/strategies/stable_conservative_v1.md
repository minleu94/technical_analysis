# 穩健策略 (Stable Conservative v1.0)

## 策略理念（Why）

### 核心思想
在均值回歸市場中，尋找被低估的機會，追求穩定報酬。這是一個低風險、穩定報酬的策略，適合風險承受度低的投資者。

### 為什麼選擇這個策略？
1. **尋找被低估的機會**：當股票價格低於其內在價值時進場
2. **穩定獲利**：不追求快速獲利，而是追求穩定的報酬
3. **避免頻繁交易**：等待明確的信號，減少交易成本

### 適用場景
- 市場處於均值回歸狀態（Reversion Regime）
- 股票價格被低估，有回歸均值的趨勢
- 投資者風險承受度低，追求穩定報酬

### 不適用場景
- 市場處於明確的上升趨勢（Trend Regime）
- 出現突破信號（Breakout Regime）
- 投資者追求快速獲利

## 風險屬性

- **風險等級**：低（low）
- **預期報酬**：中等
- **最大回撤**：較小
- **交易頻率**：較低

## 策略參數

- **買入閾值**：55（較低，尋找被低估的機會）
- **賣出閾值**：45（較低，穩定獲利）
- **買入確認天數**：3（較長的確認期，避免假信號）
- **賣出確認天數**：3（較長的確認期）
- **冷卻期**：5天（避免頻繁交易）

## 注意事項

1. **耐心等待**：此策略需要較長的確認期，不要急於進場
2. **部位大小**：可以配置較大的部位（20-30%），因為風險較低
3. **市場狀態監控**：只有在 Reversion Regime 時使用此策略
4. **避免在趨勢市場使用**：在 Trend 或 Breakout Regime 時應切換到其他策略
5. **可能錯過快速上漲**：由於確認期較長，可能錯過快速上漲的機會
