# 暴衝策略 (Momentum Aggressive v1.0)

## 策略理念（Why）

### 核心思想
在強勢趨勢市場中，追隨動能，追求快速獲利。這是一個高風險、高報酬的策略，適合風險承受度高的投資者。

### 為什麼選擇這個策略？
1. **趨勢明確時進場**：當市場出現明確的上升趨勢時，順勢而為可以獲得較好的報酬
2. **快速獲利**：不等待過多確認，快速進場以捕捉趨勢的早期階段
3. **快速止盈**：當趨勢轉弱時，快速出場以保護獲利

### 適用場景
- 市場處於明確的上升趨勢（Trend Regime）
- 出現突破信號（Breakout Regime）
- 投資者風險承受度高，追求快速獲利

### 不適用場景
- 市場處於均值回歸狀態（Reversion Regime）
- 市場波動劇烈但無明確趨勢
- 投資者風險承受度低

## 風險屬性

- **風險等級**：高（high）
- **預期報酬**：高
- **最大回撤**：可能較大
- **交易頻率**：中等偏高

## 策略參數

- **買入閾值**：70（較高，確保強勢信號）
- **賣出閾值**：50（較高，快速止盈）
- **買入確認天數**：1（快速進場）
- **賣出確認天數**：1（快速出場）
- **冷卻期**：2天（避免頻繁交易）

## 注意事項

1. **嚴格控制部位大小**：由於風險較高，建議單一部位不超過總資金的 10-20%
2. **設置停損**：建議設置 5-10% 的停損點
3. **市場狀態監控**：只有在 Trend 或 Breakout Regime 時使用此策略
4. **避免在震盪市場使用**：在 Reversion Regime 時應切換到其他策略
