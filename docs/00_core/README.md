# 核心文檔目錄

> **核心文檔（必讀）**

本目錄包含系統的核心文檔，是所有其他文檔的權威來源。

## 📄 文檔列表

- **`DEVELOPMENT_ROADMAP.md`** ⭐ **最重要**
  - 開發路線圖（最高權威）
  - Phase 狀態、功能清單、技術架構的 Single Source of Truth
  - 所有其他文件必須與 Roadmap 的 Living Section 一致

- **`PROJECT_SNAPSHOT.md`** ⭐ **開場必讀**
  - 專案快照（開場 30 秒狀態）
  - 從 Roadmap 抽出的短版入口
  - 必須與 Roadmap 的當前狀態一致

- **`DOCUMENTATION_INDEX.md`**
  - 文檔索引（文檔結構的 Single Source of Truth）
  - 完整的文檔導航和分類

- **`DOC_COVERAGE_MAP.md`**
  - 文檔覆蓋矩陣（Documentation Agent 判斷 coverage 的規則文件）
  - 定義變更類型 → 必須更新的文件對照表

- **`note.txt`**
  - 開發進度記錄
  - 詳細的開發日誌
